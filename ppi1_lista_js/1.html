O que é JavaScript?

Motivação da criação do JavaScript:

O JavaScript foi criado para dar vida às páginas web. Antes dele, os sites eram estáticos (só texto e imagens). Com JS, passou a ser possível criar interações como cliques, validação de formulários, animações e mudanças dinâmicas no conteúdo sem precisar recarregar a página.

Diferença entre execução no navegador e no servidor:

No navegador (front-end), o JavaScript roda dentro do ambiente do usuário e manipula a interface da página (HTML e CSS). Já no servidor (como no Node.js), ele roda “por trás”, lidando com banco de dados, autenticação, regras de negócio e APIs, sem depender do navegador.

Conceito de sandbox no navegador:

Sandbox é um ambiente de segurança onde o JavaScript é executado de forma isolada. Isso impede que o código tenha acesso direto ao sistema operacional do usuário, como arquivos pessoais ou configurações do computador, protegendo contra ações maliciosas.

Objetos globais como document, console e alert:

document: permite acessar e manipular elementos da página HTML (DOM).
console: usado para depuração, mostrando mensagens e erros no console do navegador.
alert: exibe uma janela simples de mensagem para o usuário na tela.
