Diferenças entre Internet e Web:

A internet é a infraestrutura global de redes conectadas, responsável por transportar dados entre dispositivos no mundo inteiro.
A web (World Wide Web) é um dos serviços que funciona sobre a internet, permitindo acessar sites e páginas por navegadores.
Ou seja, a internet é a “rede física e lógica” de conexão.
Já a web é o sistema de páginas, links e conteúdos que utilizamos dentro dessa rede.
