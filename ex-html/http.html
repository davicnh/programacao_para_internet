O que é HTTP e qual seu funcionamento?

O HTTP (HyperText Transfer Protocol) é o protocolo responsável pela comunicação entre o navegador e o servidor na internet.
Quando o usuário acessa um site, o navegador envia uma requisição HTTP ao servidor, que responde enviando páginas, imagens, vídeos e outras informações necessárias para o funcionamento do site.

Definindo o que são clientes, servidor, requisição e resposta:

Clientes: O cliente é quem faz o pedido, normalmente o navegador do usuário.

Servidor: O servidor é o computador que armazena e envia as informações do site.

Requisição: A requisição é o pedido enviado pelo cliente

Resposta: É o retorno enviado pelo servidor com os dados solicitados.

Descrição dos métodos GET e POST:

GET: É usado para solicitar informações ao servidor, como abrir uma página ou pesquisar dados.

POST: utilizado para enviar informações ao servidor, como formulários de login, cadastros e mensagens.

Exemplo de uma requisição HTTP:

GET /index.html HTTP/1.1
Host: www.site.com

Exemplo de uma resposta HTTP:

HTTP/1.1 200 OK,
Content-Type: text/html

Explicação de códigos HTTP:

  • 200 - Requisição realizada com sucesso
  • 301 ou 302 - Redirecionamento de página
  • 400 - Requisição inválida
  • 403 - Acesso proibido
  • 404 - Página não encontrada
  • 500 - Erro interno do servidor
